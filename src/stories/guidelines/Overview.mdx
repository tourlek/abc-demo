# Design System

Welcome to the **ABC 2026 CMS Design System**. This design system provides a comprehensive suite of UI components, guidelines, and assets to build consistent and accessible user interfaces.

## Structure

The Storybook is organized into the following sections:

- **Guidelines**: Core design principles, typography, colors, and spacing.
- **Example**: Example application components and page layouts.
- **UI**: Individual UI components (atoms and molecules) from the shadcn/ui library.

## Getting Started

Explore the **Guidelines** section to understand the foundational elements of the design system. Then, browse the **UI** section to see individual components in action.

## Technologies

- **React**: UI library
- **Tailwind CSS**: Styling framework
- **Radix UI**: Accessible primitives
- **shadcn/ui**: Re-usable component library
